# Within Canva asset names

Export final Canva assets as transparent PNGs where possible, then place them in this folder.

The website is already wired to look for these filenames:

## Core brand
- `within-logo-mark.png` — small logo mark for the navigation circle
- `within-logo.png` — full logo or logo concept for the footer
- `within-logo-concept.png` — soft watermark logo concept for the booking section

## Main artwork
- `within-hero.png` — hero artwork, preferably transparent PNG, abstract face / botanical / neuroaffirming brand image
- `within-section-divider.png` — soft botanical divider or background accent for the About section
- `within-approach-art.png` — artwork for the approach section

## Service icons
- `within-icon-women.png`
- `within-icon-parenting.png`
- `within-icon-neurodivergent.png`
- `within-icon-advocacy.png`

## Resource preview images
- `within-resource-big-feelings.png`
- `within-resource-school-meeting.png`
- `within-resource-family-rhythm.png`

Notes:
- Use PNG with transparent background for logos and icons.
- Avoid Canva-generated text inside images unless it is part of the logo.
- Keep large artwork under roughly 2000px wide if possible.
- If an image is missing, the website will hide it and keep the existing fallback design.
