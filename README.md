# Within Studio

A Vite + React prototype for creating neuroaffirming therapy resources, Explorer Collection child books, parent scripts and printable activity pages.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Current build

- Interactive Book Creator cards
- Explorer Collection starter resource library
- Parent support script generator
- Printable resource preview pages
- Copy-to-clipboard generated text
- Mobile responsive styling
